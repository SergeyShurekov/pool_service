## Итоговая схема сайта

```
/                           — Главная (УТП, услуги, калькулятор, портфолио)
/obsluzhivanie-basseynov/   — Обслуживание бассейнов (прайс, список работ, типы)
/obsluzhivanie-basseynov-dlya-chastnyh-domov/ — Для частных домов и малых гостиниц
/remont-oborudovaniya-basseynov/ — Ремонт и обслуживание оборудования
/himicheskaya-obrabotka-vody/    — Химия, хлор, водоподготовка (в т.ч. запросы про хлорку)
/basseyn-bez-hlora/             — Бассейн без хлора (УТП, технологии)
/portfolios/                    — Портфолио / кейсы
/kalkulyator/                   — Калькулятор стоимости обслуживания
/kontakty/                      — Контакты (Сочи, адрес, карта, форма)
```
